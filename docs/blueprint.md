# **App Name**: JustTry FinTech CRM

## Core Features:

- Role-Based Dashboards: Separate dashboards for Sales, Back Office, and Admin users, each with access to relevant modules and functionalities. Implement secure authentication and role-based routing.
- Lead Management: Add, edit, delete, and assign leads with comprehensive fields including personal information, service type, financial details, and document uploads. Include notes and follow-up tracking.
- Dynamic Service Pipelines: Create dynamic pipeline stages based on the selected service type (Loan, Investment, Insurance), ensuring a tailored workflow for each lead.
- Automated Task Creation: Use a tool that analyzes lead data and status to create a list of required tasks for the user based on the pipeline the lead is in.
- Contact & Company Management: Store client information including contact details, employment, service history, and linked company details for corporate leads. Provide easy access and organization of client data.
- Document Management: Upload and verify KYC documents, bank statements, and other relevant files. Streamline document processing and ensure compliance.
- Analytics & Dashboard KPIs: Display key performance indicators (KPIs) relevant to each service type (Loan, Investment, Insurance) on the admin dashboard. Track total applications, approval rates, active SIPs, and policies issued.

## Style Guidelines:

- Primary color: A professional, trustworthy blue (#29ABE2) to inspire confidence and reliability.
- Background color: A light, desaturated blue (#E5F6FD) for a clean and calming backdrop.
- Accent color: A complementary green (#90EE90) to highlight key actions and positive outcomes.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its neutral and modern appearance.
- Use clear, consistent icons to represent service types, actions, and status indicators.
- Maintain a clean and organized layout with intuitive navigation. Prioritize key information for each user role.
- Incorporate subtle animations to provide feedback on user actions and guide them through workflows.
